# 附录B：常见问题

## 1. 扩展开发基础问题

### 1.1 如何开始开发扩展？
**问题**: 我是新手,想开发扩展但不知道从何开始?

**解答**: 
1. 首先阅读本教程第一章,了解基础开发环境搭建
2. 创建一个最简单的扩展,包含一个武将和一个技能
3. 按照教程逐步学习,循序渐进
4. 参考其他扩展的源码学习

### 1.2 扩展文件结构
**问题**: 扩展需要哪些必要文件?

**解答**:
基本文件结构:
```
extension/
  └── my_extension/
      ├── extension.js     # 必需,扩展主文件
      ├── character.js     # 可选,武将代码
      ├── card.js         # 可选,卡牌代码
      ├── audio/          # 可选,音频文件
      └── image/          # 可选,图片文件
```

### 1.3 代码编辑器选择
**问题**: 用什么编辑器开发比较好?

**解答**:
1. 推荐使用VS Code:
   - 免费开源
   - 插件丰富
   - 支持代码高亮
   - 集成终端
   - 调试功能强大

2. 其他选择:
   - Sublime Text
   - Notepad++
   - WebStorm

## 2. 武将开发问题

### 2.1 武将属性设置
**问题**: 武将的基本属性如何设置?

**解答**:
```javascript
"my_general": ["male", "shu", 4/4, ["my_skill"], [
    "des:武将描述",
    "ext:扩展名/image.jpg"
]],
```
- 第一项: 性别(male/female)
- 第二项: 势力
- 第三项: 体力值
- 第四项: 技能列表
- 第五项: 特殊标记

### 2.2 技能无法触发
**问题**: 技能定义了但无法触发?

**解答**:
检查以下几点:
1. 触发时机是否正确
2. filter函数返回值是否为true
3. content函数是否正确
4. 技能是否正确添加到武将
5. 控制台是否有报错

### 2.3 联机兼容性
**问题**: 如何确保扩展可以联机?

**解答**:
1. 添加connect属性:
```javascript
name: 'my_extension',
connect: true,
```

2. 使用标准API
3. 避免本地特殊效果
4. 正确处理同步

## 3. 技能开发问题

### 3.1 技能类型选择
**问题**: 不同类型技能如何选择?

**解答**:
1. 触发技能:
   - 在特定时机自动触发
   - 使用trigger定义触发条件

2. 主动技能:
   - 玩家手动选择发动
   - 使用enable定义发动时机

3. 视为技能:
   - 将一张牌当做另一张牌使用
   - 使用viewAs定义转化效果

### 3.2 技能调试方法
**问题**: 如何调试技能效果?

**解答**:
1. 使用console.log输出信息:
```javascript
content: function(){
    console.log('当前玩家:', player);
    console.log('目标数量:', targets.length);
}
```

2. 使用game.print在游戏中显示信息
3. 检查浏览器控制台错误信息
4. 使用断点调试

### 3.3 性能优化
**问题**: 技能效果卡顿怎么办?

**解答**:
1. 减少不必要的循环
2. 使用缓存机制
3. 优化判断条件
4. 减少事件触发
5. 合理使用异步操作

## 4. 资源管理问题

### 4.1 图片问题
**问题**: 图片显示不出来?

**解答**:
1. 检查图片路径是否正确
2. 确认图片格式(jpg/png)
3. 图片尺寸是否合适
4. 文件名是否正确
5. 区分大小写

### 4.2 音频问题
**问题**: 技能音频无法播放?

**解答**:
1. 检查音频文件格式(mp3)
2. 确认音频路径正确
3. 音频命名是否规范
4. 检查audio属性设置
5. 确认文件存在

### 4.3 资源优化
**问题**: 扩展文件太大怎么办?

**解答**:
1. 压缩图片资源
2. 优化音频质量
3. 删除未使用文件
